# html로 표 만들기

## 태그들

### \<table>

표 전체의 윤곽을 잡는다

### \<tr>

행을 만든다

### \<td>

셀을 만든다

### \<th>

제목 셀을 만든다

### \<colspan>

여러 셀을 가로로 하나로 합친다

```html
<td colspan="합칠 셀의 개수">내용</td>
```

### \<rowspan>

여러 셀을 세로로 하나로 합친다

```html
<td rowspan="합칠 셀의 개수">내용</td>
```

### \<caption>

표에 제목을 붙인다

```html
<caption>표 제목</caption>
```

### \<figcaption>

표에 제목을 붙인다

```html
<figcaption>
<p>제목을 이런 식으로도 붙일 수 있다니!! 와대박</p>
</figcaption>
```
