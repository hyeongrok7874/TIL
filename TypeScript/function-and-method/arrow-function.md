# 화살표 함수

```ts
const 함수 이름 = (매개변수1: 타입1, 매개변수2: 타입2[, ...]) : 반환 타입 => 함수 몸통
```

함수 몸통의 중괄호를 사용 시 실행문 방식 몸통, 비 사용 시 표현식 문 방식 몸통이 된다

## 실행문

CPU에서 실행된 결과를 return 키워드를 사용해야 알 수 있다

### 복합 실행문

중괄호를 사용하여 감싼다

컴파일러로 하여금 여러 실행문을 한 개처럼 인식하게 한다

변수의 유효 범위도 지역 범위로 제한한다

### 함수 몸통과 복합 실행문

function 키워드로 만드는 함수는 반드시 몸통을 중괄호로 감싸야 하는데, 여기서 중괄호는 앞서 설명한 복합 실행문이다

따라서 함수 몸통은 여러 줄로 구현할 수 있다

### return 키워드

CPU에서 실행된 결과를 반환받기 위한 키워드

반드시 함수 몸통(복합 실행문)에서만 사용할 수 있다

## 표현식 문

CPU에서 실행된 결과를 굳이 return 키워드를 사용하지 않아도 값이 반환된다

### 표현식 문 스타일의 화살표 함수

실행문

```ts
const isGreater = (a: number, b: number): boolean => {
  return a > b;
}
```

표현식

```ts
const isGreater = (a:number, b: number): boolean => a > b
```

표현식은 값을 반환하는 실행문으로 return 키워드가 생략되었다


## 세미콜론

ES5 JS의 모든 문장 끝에 세미콜론이 있어야 한다

반면에 ESNext JS와 TS에서는 세미콜론을 생략할 수 있다

TS에서는 관습적으로 표현식 문에는 세미콜론을 붙이지 않는다
