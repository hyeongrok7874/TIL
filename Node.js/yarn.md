# yarn

facebook에서 만든 새로운 JS 패키지 매니저

npm과 동일한 기능을 수행한다

## 사용법

### 프로젝트 초기화/시작

```
$ yarn init
```

### 패키지 설치

- package.json 의존성 모듈

```
$ yarn 
# or
$ yarn install
```

- 의존성 모듈

```
$ yarn add [package]
$ yarn add [package]@[version]
$ yarn add [package]@[tag]
```

### 패키지 제거

```
$ yarn remove [package]
```

### 패키지 업그레이드

```
$ yarn upgrade [package]
```
