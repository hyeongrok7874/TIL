# 하드웨어로서의 서버

## pc와의 구성 차이

> 서버는 전원을 끄지 않는다.

### 구성상의 차이

> 신뢰성이 높을 뿐만 아니라 높은 가용성도 가지고 있습니다.

## PC와의 성능 차이

### 요구되는 성능 차이

> 일반 pc는 표시 성능이 중요하다.<br>
> 서버는 I/O 성능이 중요하다.

## 서버 OS

### 종류

> - Window Server
> - Linux
> - UNIX 계열

### 각 OS의 장점

#### Window Server

> 사용자 인터페이스로 조작할 수 있다.<br>
> 기업이나 단체에서 필요로 하는 기능이 미리 패키지로 되어 있고 MS의 지원도 있다.

#### Linux

> 비교적 간단하고 저렴하게 시스템을 구축할 수 있다.
